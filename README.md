**Herramienta todo-en-uno para cálculo y conversión de resistencias THT y SMD**
**Calculadora de resitencias THT y SMD asi como de ley de Ohm , con capacidad para exportar tus ejecuciones en distintos formatos .**

*Software profesional desarrollado por PIXELBITS Studio que simplifica el trabajo con resistencias mediante cálculos precisos, visualización gráfica y exportación de resultados.*

---

## Descarga y Ejecución Inmediata

1. **Descargue** el archivo ZIP
2. **Extraiga** en cualquier carpeta
3. **Ejecute** `resistencialSpotLITE.exe`
4. **Comience** a calcular al instante

**No requiere instalación de Python o dependencias adicionales**
